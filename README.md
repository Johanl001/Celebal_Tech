## 📌 Week 1 Highlights — Statistics, Probability & Linear Algebra

| Topic | Description |
|:------|:------------|
| 🧹 **Data Cleaning & Preprocessing** | Handling missing values, outliers, and data types |
| 📊 **Descriptive Statistics** | Mean, median, mode, variance, standard deviation |
| 🔍 **Inferential Statistics** | Confidence intervals, hypothesis testing, p-values |
| ✅ **Hypothesis Testing** | t-tests, chi-square tests, ANOVA |
| 🔗 **Correlation Analysis** | Pearson, Spearman correlation coefficients |
| 📈 **Concept Drift & PSI** | Population Stability Index for model monitoring |
| 🎲 **Probability Theory** | Distributions, conditional probability, joint probability |
| 🔮 **Bayes' Theorem** | Prior, posterior, and likelihood |
| 📐 **Central Limit Theorem** | Sampling distributions and normality |
| 🔢 **Eigenvalues & Eigenvectors** | Matrix decomposition fundamentals |
| 🔄 **SVD** | Singular Value Decomposition for dimensionality reduction |
| 🗜️ **PCA** | Principal Component Analysis for feature compression |

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/dhanrajdeshmukh/data-science-ml-learning-journey.git
cd data-science-ml-learning-journey
```

### 2. Set up a virtual environment

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```
